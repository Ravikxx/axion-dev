What it does

Everything you need to build faster

Axion brings together the tools that actually matter for an AI coding agent.

⌨️

Terminal Agent

Full agentic loop in your terminal. Edit files, run commands, fix bugs — Claude-Code style, open source.

🖥️

Computer Use

Vision + screenshot analysis lets the agent see and click your screen, not just your code.

🌐

Chrome Extension

Read pages, click elements, fill forms, and automate the browser — directly from the agent.

💬

Discord Integration

Run the agent as a Discord bot. Queue tasks, cancel runs, and get responses in any channel.

🔌

MCP Servers

First-party MCP servers for Blender and more. Plug in any tool with the Model Context Protocol.

⚡

Multi-model

Switch between Claude, Lumen, Veil, or any compatible model. Use the best tool for each task.

Quick start

Up and running in minutes

Clone the repo, install dependencies, and start your first session.

# Clone and install
$ git clone https://github.com/Ravikxx/axion
$ cd axion && npm install

# Set your API key and run
$ export ANTHROPIC_API_KEY=sk-...
$ npm start
